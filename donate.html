Configuração dos Stripe Payment Links

Para ativar os pagamentos via cartão de crédito:

  1. Acesse o Dashboard do Stripe
  2. Vá em "Payment Links" no menu lateral
  3. Crie links para cada valor (R$10, R$30, R$50, R$100)
  4. Crie links separados para assinaturas mensais
  5. Crie links com "Allow customer to adjust quantity" para valores personalizados
  6. Substitua os links "https://buy.stripe.com/test_..." pelos seus links reais
